# Changelog

All notable changes to this project will be documented in this file.

The format is based on [Keep a Changelog](https://keepachangelog.com/en/1.0.0/),
and this project adheres to [Semantic Versioning](https://semver.org/spec/v2.0.0.html).

## [0.1.0] - 2026-04-18

### Added
- Initial release with AI-powered tagging
- Native GUI built with iced 0.13
- Three theme options: Light, Dark, and Eldritch
- Full-text search with SQLite FTS5
- Real-time file monitoring with notify
- Taxonomy system for pending/approved tags
- Auto-update checking via GitHub Releases API
- Daemon-based background processing

### Features
- File browser with grid and list views
- Tag-based filtering
- Category organization (documents, images, code, audio, video)
- Pending tag approval queue
- Approved tag library with categories and aliases
- Settings panel for Ollama configuration
- First-run setup wizard
